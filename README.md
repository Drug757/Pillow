# Pillow 

## 1. Краткое описание инструмента
Pillow — это библиотека Python для работы с изображениями.

## 2. Цель изучения
Изучить базовые операции с изображениями.

## 3. Описание реализованной задачи
Реализована программа, которая открывает изображение, изменяет его размер и сохраняет результат.

## 4. Инструкция по запуску
1. Установить библиотеку:
   ```
   pip install pillow
   ```
2. Запустить:
   ```
   python main.py
   ```

## 5. Общие выводы
Библиотека удобная и простая для работы с изображениями.
