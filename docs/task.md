# Техническое задание

## 1. Описание задачи
Разработать программу, которая демонстрирует базовые возможности библиотеки Pillow для обработки изображений.

## 2. Цель работы
Изучить основные операции с изображениями: открытие, изменение размера и сохранение.

## 3. Функциональные требования
Программа должна:
- открыть изображение из файла  
- изменить его размер  
- сохранить результат в новый файл  

## 4. Нефункциональные требования
- код должен быть понятным  
- программа должна работать быстро  
- результат должен корректно сохраняться  

## 5. Ограничения и допущения
- используется одно изображение  
- без сложных эффектов  
- работа только с файлами
